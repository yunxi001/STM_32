# 二维自由度舵机云台

#### Description
采用舵机型号为DS3218，二维自由度的舵机云台

1、使用MPU6050读取翻滚角Roll和横向角Yaw，控制二维舵机
2、使用ADC采样摇杆模块的X轴和Y轴，控制云台的运动
3、扩展，根据OpenMV识别物品(如色块、物品、数字等)传回的数据，控制二维舵机的运动，激光笔接于舵机上，准确打到物品上
